## Roles and responsibilities

### Backend Engineer
Develops server-side logic, APIs, and business services. Ensures services are scalable, reliable, and perform efficiently under high load.

### DevOps Engineer
Maintains cloud infrastructure, configures CI/CD pipelines, monitors system health, and automates deployment processes.

### QA Engineer
Designs and executes test cases, performs functional and integration testing, and ensures system reliability and product quality.

### Database Engineer
Designs and optimizes database schemas, ensures data consistency, manages indexing strategies, and improves query performance.

### Search Engineer
Develops and maintains search indexing systems, implements ranking and filtering algorithms, and optimizes search performance.


## Common skills across roles

- Programming languages such as Java, Kotlin, Go, Python, or JavaScript
- Understanding of distributed systems and microservice architecture
- Experience with REST APIs and asynchronous messaging systems
- Knowledge of relational and NoSQL databases
- Familiarity with cloud infrastructure and containerization technologies
- Experience with version control systems such as Git
- Understanding of monitoring, logging, and performance optimization
- Knowledge of software testing methodologies
